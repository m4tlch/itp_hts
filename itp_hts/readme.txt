ITP Hierarchical Taxonomy Select

Developer: Kirill Danilevsky, k.danilevsky@gmail.com
web: http://itprostor.com

This is simple widget with own formatter that provide fast build hierarchical select.

High speed provided direct access to DB with help Ajax.

Install module.

In section Manage field (Content types) add new taxonomy_term_reference field and add widget "ITP hierarchical taxonomy select"

After this select Number of values equal Unlimited (Edit field) and fill Name select field.

In section Manage display you can select two types formatter: ITP hierarchical plan text or ITP hierarchical link.

And in this section you can set delimiter symbol that will split your taxonomy terms.
